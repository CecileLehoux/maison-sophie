## Priorités SEO
- Balises HTML sémantiques (`<header>`, `<main>`, `<nav>`, `<section>`, `<article>`, `<footer>`)
- Meta title/description uniques par page
- Balises Open Graph et Twitter Card
- Schema.org JSON-LD (LocalBusiness, BeautySalon)
- Core Web Vitals : LCP < 2.5s, CLS < 0.1, FID < 100ms
- Images avec `alt` descriptifs, format WebP, lazy loading
- URLs canoniques propres
- Sitemap XML + robots.txt
